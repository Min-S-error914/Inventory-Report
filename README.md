# 📊 Inventory Report System (Excel)

## 📌 Project Title
ABC Retail Mart - Inventory Report System

## 📅 Report Period
February 2026

---

## 📖 Description
This project is an Inventory Management System built using Microsoft Excel. It tracks stock in, stock out, unit price, and total inventory value.

---

## 🚀 Features
- Inventory tracking (Stock In / Stock Out)
- Automatic total value calculation
- Monthly report generation
- Summary statistics:
  - Total Items
  - Total Stock In
  - Total Stock Out
  - Total Inventory Value
  - Highest & Lowest Value
- Clean tabular format

---
